# Button

Chakra inspired Button.

```jsx
<Button>Default</Button>
```

## Usage

### Outline variant

```jsx
<Button variant="outline">Outline</Button>
```

### Orange

```jsx
<Button color="DSOrange">Outline</Button>
```
